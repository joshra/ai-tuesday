/02 About

什麼是
AI 星期二？

一個低壓、固定、技術導向的午間交流聚會，讓工程師、產品人與研究者可以分享自己最近實際嘗試的 AI 工具、專案、工作流與產品觀察，並透過討論累積可重複參考的經驗。

不需要完整簡報，也不需要包裝成正式演講。好的分享可以很短，重點是你實際試過什麼、結果如何、哪裡有效、哪裡不好用。

/04 Positioning
AI 星期二是 ——
  • 低壓、固定、技術導向的午間交流聚會
  • 分享實作經驗、踩坑紀錄與工具評估的地方
  • 工程師、產品人、研究者的水平對話空間
  • 可以逐步累積、公開整理的共筆社群
  • 可以穩定預期、每週出現的技術節奏
不是 ——
  • 課程、行銷場或商業推銷群
  • 大量未消化新聞的轉貼群組
  • 沒有脈絡的工具清單堆疊
  • 過度空泛的 AI 趨勢討論
  • 與 AI 技術或實務無關的閒聊頻道
/05 Agent-Native

人決定方向，
Agent 做例行工作。

AI 星期二把自己設計成一個 agent 原生社群：人負責建立信任、判斷方向、做最終決定；agent 負責整理共筆、歸納主題、準備流程、追蹤進度。

這不是自動化實驗，而是把每次 meetup 的例行工作盡量交給 agent 處理，讓人的注意力留在實質的技術討論上。

AGENT / 01
主持助理
整理議程草稿、生成開場提示詞、追蹤分享者確認與會後提醒。
AGENT / 02
共筆整理
將會後紀錄結構化為 Markdown，標記主題標籤，公開前做敏感資訊檢查。
AGENT / 03
策展 Agent
歸納每週主題方向、分類成員分享的連結與工具，產出週報摘要草稿。
AGENT / 04
新人導覽
歡迎新成員加入、推薦適合的過往共筆、協助重寫問題讓討論更有焦點。
AGENT / 05
分享準備
協助講者從粗筆記整理出 15 分鐘的大綱結構，預測可能的提問。
AGENT / 06
知識庫 Agent
對歷次共筆做主題索引與語意搜尋，輸出主題地圖與學習路徑建議。

加入 AI 星期二

我們有一個小型 AI 技術分享群「AI星期二」，固定每週二中午線上聊 50 分鐘。主題偏實作 — AI coding、agent workflow、工具實測、論文導讀與產品觀察。這不是課程，也不是行銷場，主要是工程與產品實務交流。如果你對 AI 工具或應用實作有興趣，歡迎一起來聽或偶爾分享。
